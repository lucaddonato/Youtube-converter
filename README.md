# Youtube converter
Download and convert YouTube videos to the following formats:

### 🎥 Vídeo
- .mp4
- .mov
- .avi

### 🎵 Áudio
- .mp3
- .wav
- .flac

## How to use?
1. Paste the video link
2. Choose where to save the file
3. Pick the format you want
4. That's it! 🎉

To run the app, just open the file:
**main.py**
